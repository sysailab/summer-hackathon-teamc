# 요구사항
1. output.csv 형태로 결과를 출력함
2. 논문의 저자를 구분하여 각각 표기해야 함
3. 출력 파일의 형태는 다음과 같음

|저자|소속대학|논문제목|사사표기여부|

# 예제
- 저자: 최창범1, 장수영1, Jane Doe2
- 소속: 1한밭대학교 2한국대학교
- 제목: 세상을 변화시키는 방법
